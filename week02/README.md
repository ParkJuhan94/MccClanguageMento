### 변수 연산
```c
/*
연산자의 종류
+ - * / % =
+= -= *= /= %=
++ --  (전치 또는 후치)
*/
```

```c
int a = 5;

a += 6;   // a = a + 6;
a *= 7;   // a = a * 7;
```

```c
int a = 10;
int b;

//후치 증가 연산
b = a++;
printf("a : %d\n", a);    // a : 11
printf("b : %d\n", b);    // b : 10

//b에 a가 대입된 후에-> a가 증가
```
